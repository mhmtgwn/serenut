Serenut OS İndirme Merkezi

Windows bilgisayarlar ve Android el terminalleri için optimize edilmiş stabil terminal yazılımlarımız.

💻 Windows Sistem Gereksinimleri

  • • Windows 10 veya Windows 11 (64-bit)
  • • Minimum 4 GB RAM bellek (8 GB önerilir)
  • • 1.6 GHz çift çekirdekli veya daha hızlı işlemci
  • • ESC/POS uyumlu 58mm / 80mm Termal fiş yazıcısı sürücüsü

📱 Android El Terminali Gereksinimleri

  • • Android 7.0 (Nougat) veya üzeri işletim sistemi
  • • Minimum 2 GB RAM (Dahili termal yazıcılı Sunmi, Urovo cihazları desteklenir)
  • • Bluetooth 4.0 / USB bağlantı destekli taşınabilir yazıcılar
